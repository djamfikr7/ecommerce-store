# E-Commerce Web Store

## What This Is

A full-featured, modern e-commerce platform designed for mid-market scale (~10K products) with integrated social media automation. Built with a dark gradient neomorphic design system featuring 3D animated buttons and full localization support. The platform focuses on exceptional product discovery, seamless checkout experience, comprehensive admin dashboard, user engagement features, and automated social media integration for multi-platform marketing, lead tracking, automated posting, and comment management.

## Core Value

A visually stunning, performant e-commerce experience that converts browsers into buyers through intuitive product discovery and frictionless checkout, while providing merchants with powerful tools to manage their business and automate their social media marketing campaigns.

## Tech Stack

- **Framework:** Next.js 15.x + React 19.x + TypeScript 5.6+
- **Database:** PostgreSQL 16.x + Prisma 6.x
- **Styling:** Tailwind CSS 3.4+ + Framer Motion 12.x + @react-three/fiber 8.x
- **Payments:** Stripe 17.x
- **Auth:** Next-Auth 5.x (Auth.js)
- **Testing:** Vitest 2.x + Playwright 1.48+
- **CI/CD:** GitHub Actions + Vercel + Supabase
